=== EBNF rule

Exact gramatical description can be provided using EBNF rule.

.EBNF rule table (several variants)
[header]
|=====
|Usage		|Notation
|definition	|`:` or `::=` or `:=` or `=`
|concatenation	|`,`
|termination	|`;`
|alternation	|`\|`
|option		|`[ ... ]`
|repetition	|`{ ... }`
|grouping	|`( ... )`
|terminal string|`' ... '` or `" ... "`
|repeat 0 or more	|`*` (postfix)
|repeat 1 or more	|`+` (postfix)
|exception	|`-`
|=====
